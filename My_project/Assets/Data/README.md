# Game Database

- Редактируй `game.db` в любом SQLite GUI (DB Browser, DBeaver).
- Таблицы: `Weapons` и `ShipTemplates`. Структуру можно менять вручную.
- После правок просто сохрани файл и верни его в Unity. Чтобы увидеть изменения в рантайме, вызови `GameDatabase.InvalidateCache()` или перезапусти сцену.
